# Contributing

## Setup
- Install Godot `4.5.1`.
- Open/import `project.godot` and run from the editor.

## Branching
- Use `feature/` or `bugfix/` branches (avoid working directly on `main`).

## Project Notes
- Avoid relying on GDScript type inference for non-trivial values; prefer explicit types.
- Export guidance lives in `EXPORTING.md`.
